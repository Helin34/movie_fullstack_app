# 🎬 Film Yönetim Platformu

Bu proje, React (Frontend) ve Node.js (Backend) kullanarak geliştirmiş olduğum bir Fullstack film yönetim uygulamasıdır.

## 🚀 Proje Özellikleri

- 🎞️ Film listesini görüntüleme
- ➕ Yeni film ekleme
- 🗑️ Film silme
- 🔍 Detay sayfası

## 🎥 Proje Önizlemesi

![Demo](./frontend/public/demo.gif)

## 🛠️ Kullanılan Teknolojiler

- **Frontend:** React, Tailwind CSS, Axios, TanStack Query
- **Backend:** Node.js, HTTP Module, JSON (Data)
- **Diğer:** React Router, React Icons
